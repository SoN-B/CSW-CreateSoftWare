# DUNGEON 프로젝트

>**주제** : C++ 기반 RPG 턴제 게임

- 총 3개의 맵으로 구성되어, 맵 상의 몬스터들과 전투해 나가는 방식입니다.
- 하나의 맵은 총 5개의 방으로 이루어져 각각 1~2마리의 몬스터가 등장합니다.
- 마지막 5번째 방에는 보스가 등장합니다.
